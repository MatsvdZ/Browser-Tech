# Browser Tech
Mijn site voor de minor Web Design en Development voor het vak Browser Technologies

## Eerste checkout (16-02)

### Wat heb ik vandaag gedaan?
We begonnen met een algemene uitleg over de opdracht, wat werd gevolgd door een kleine workshop over de HTML van forms. We hebben geleerd over verschillende attributes, fieldsets, legends, labels en hoe we dit op een nette manier allemaal kunnen structureren.

Dit hebben we echt uit kunnen testen door het toe te passen op de aangifte erfbelasting, die we na moesten maken, met de nieuwe HTML.

Na een tijd hier aan gezeten te hebben, moesten we een artikel lezen over icons van de nieuwe Tahoe update van MacOS, en hierbij notities gemaakt. 

Verder heb ik alvast even snel gekeken naar de styleguide van de NS, om zo alvast te kijken naar hoe de stijl eruit ziet, en even voor mezelf kon bedenken hoe ik dit mogelijk zou kunnen toepassen.


### Hoe lang duurde het?
We hebben ongeveer 2 uur aan theorie gehad over de stof, en de opdracht die we gaan doen.

Ik heb in totaal zo'n 2 uur aan de HTML zelf heb gezeten, en heb nu dus een redelijke basis neergezet voor het project in de HTML
